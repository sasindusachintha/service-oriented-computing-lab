# Service Oriented Computing (SOC)

## 📌 Overview
This repository contains practical implementations of Service Oriented Computing (SOC) concepts using RESTful APIs and service-based architecture.

## 🚀 Features
- REST API development
- Service-oriented architecture examples
- CRUD operations
- Authentication using JWT
- Modular service design

## 🛠 Technologies Used
- ASP.NET Core Web API
- Entity Framework Core
- SQL Server
- JWT Authentication

## 📂 Structure
- Lab1: Basic Service API
- Lab2: REST API implementation
- Lab3: Authentication service
- Lab4: Campus service system

## 🔐 Concepts Covered
- Service Oriented Architecture (SOA)
- RESTful services
- Stateless communication
- Decoupled system design

## 📡 Example Endpoints
### Students
- GET /api/students
- POST /api/students

### Auth
- POST /api/auth/login
